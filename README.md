# AI Powered Customer Support Service Automation

## Overview

This project is an AI-powered customer support system built on Salesforce Service Cloud. It automates customer case management using Record-Triggered Flows, queue-based routing, SLA handling, and Agentforce AI to improve support efficiency and reduce manual effort.

## Features

* Customer, Merchandise, and Order Management
* User Case Management
* Automatic Case Classification (Damage/Warranty)
* Queue-Based Case Assignment
* SLA & Priority Automation
* Agentforce AI Assistance
* Reports and Dashboards
* Lightning App Interface

## Technologies Used

* Salesforce Service Cloud
* Salesforce Flow Builder
* Lightning App Builder
* Agentforce AI
* Prompt Builder
* Reports & Dashboards

## Project Workflow

1. Customer raises a support request.
2. A User Case record is created.
3. The case is classified as Damage or Warranty.
4. The case is assigned to the appropriate queue.
5. High-priority cases are reassigned automatically based on SLA.
6. Agentforce AI provides suggestions to support agents.
7. Reports and dashboards monitor performance.
